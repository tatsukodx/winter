本プロジェクトは、Undertale サンズ戦の演出を Python（Pygame）で再現することを目的とした制作途中のゲームシステムです。現在は、ガスターブラスターの挙動とソウル（プレイヤー）の基本操作が実装されており、戦闘シーンの基盤が整いつつあります。
ガスターブラスターは、出現 → 開口 → 発射 → 後退という一連の動作をアニメーション付きで再現しています。発射時には白色のビームが口から伸び続け、ガスターブラスター本体は反動で後方へ加速しながら画面外へ退場します。また、複数のガスターブラスターを時間差で召喚できるように、専用のマネージャークラスを実装しており、(発射座標, 発射方向, 発射時間ms) を指定して任意のタイミングで出現させることが可能です。
さらに、画面下部には白いバトルボックスを配置し、その内部でソウル（sprite/soul/soul.png）を操作できるようになっています。ソウルは矢印キーで移動でき、バトルボックス外へ出ないよう制限されています。今後は、攻撃パターンの追加、ソウルの当たり判定、バトルボックスの移動やソウルの色変化など、サンズ戦特有のギミックを順次実装していく予定です。
