本プロジェクトは、Undertale サンズ戦の演出を Python（Pygame）で再現することを目的とした制作途中のゲームシステムです。現在は、ガスターブラスター（ビーム）の挙動とソウル（プレイヤー）の基本操作が実装されており、戦闘シーンの基盤を作っている状態です。
ガスターブラスターは、出現 → 開口 → 発射 → 後退という一連の動作をアニメーション付きで再現しています。発射時には白色のビームが口から伸び続け、ガスターブラスター本体は反動で後方へ加速しながら画面外へ退場します。また、複数のガスターブラスターを時間差で召喚できるように、専用のクラスを実装しており、(発射座標, 発射方向, 発射時間(ms)) を指定して任意のタイミングで出現させることができます。
さらに、画面下部にはバトルボックス(白い枠線)を配置し、その内部でソウルを操作できるようになっています。ソウルは矢印キーで移動でき、バトルボックス外へ出ないよう制限されています。今後は、攻撃パターンの追加、ソウルの当たり判定、バトルボックスの移動やソウルの色変化など、サンズ戦特有のギミックを順次実装していく予定です。
